# dotfiles

This repo contains all of the dotfiles to set up my development experience on a new machine.

## Tools
- `aerospace` (window manager)
- `ghostty` (terminal)
- `wezterm` (terminal)
- `zsh` (shell)
- `tmux` (terminal multiplexer)
- `neovim` (editor)

## Requirements (wip)
- `homebrew` (package manager)
- `fzf` (fuzzy finder)
